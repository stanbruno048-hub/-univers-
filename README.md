🌍 Univers Chat

📱 Description

Univers Chat est une application de messagerie simple créée en HTML, CSS et JavaScript.

Elle permet de :

- 👤 S’identifier (nom + numéro)
- 📱 Ajouter des contacts
- 💬 Discuter avec des contacts
- 💾 Sauvegarder les messages (localement)
- 🔐 Accéder à une zone secrète avec un code

---

📂 Structure du projet

index.html   → structure de l’application
style.css    → design (interface)
script.js    → logique (fonctionnalités)

---

🚀 Installation

1. Crée un dossier sur ton téléphone ou ordinateur
2. Ajoute 3 fichiers :

index.html
style.css
script.js

3. Copie le code dans chaque fichier
4. Ouvre "index.html" avec ton navigateur

---

▶️ Utilisation

1. Lance l’application
2. Clique sur Entrer
3. Remplis ton nom et ton numéro
4. Ajoute un contact
5. Clique sur le contact pour discuter

---

🔐 Zone secrète

- Accès via bouton 🔐 Secret
- Code par défaut :

1234

---

⚠️ Limites

- ❌ Pas de connexion internet
- ❌ Pas de vrais contacts du téléphone
- ❌ Chat local uniquement

---

🔥 Améliorations possibles

- 📷 Envoi d’images
