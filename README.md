# 💬 Application de Messagerie Universelle

Bienvenue dans **MonAppChat**, une application de messagerie simple, rapide et moderne inspirée de WhatsApp.

---

## 📱 Fonctionnalités

- 🔐 Authentification des utilisateurs
- 💬 Envoi et réception de messages en temps réel
- 👥 Accès aux contacts
- 🟢 Statut en ligne / hors ligne
- 📩 Notifications de nouveaux messages
- 📁 Envoi de fichiers (images, documents)
- 🎨 Interface moderne et responsive

---

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- (Optionnel) Node.js pour le serveur
- (Optionnel) Base de données locale ou API
